1. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

CSS mempercantik tampilan web, JavaScript menambahkan interaksi dinamis seperti event, animasi, dan logika aplikasi.

2. Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web?

Dapat disisipkan langsung dalam file HTML menggunakan tag <style> dan <script>, atau melalui file eksternal.

3. Jelaskan apa saja keuntungan menggunakan JavaScript dan CSS?

JavaScript memberikan fungsionalitas interaktif, sedangkan CSS mengatur tampilan layout dan desain halaman.

4. Jelaskan apa saja keuntungan menggunakan Bootstrap?

Bootstrap memberikan framework CSS siap pakai yang memudahkan pengembangan web responsif dan modern.

5. Contoh Penggunaan Bootstrap

Contoh: <button class="btn btn-primary">Klik Saya</button>
Fungsinya: Membuat tombol dengan gaya Bootstrap.
